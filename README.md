# GKID Kernel

A GKI kernel featuring SukiSU Ultra and SUSFS support.

## Features
- SukiSU Ultra built-in
- SUSFS integration
- Optimized for performance & battery life
- Regular updates

## Compatibility
- Built for Poco X6 Pro (Duchamp)
- Works on any device with **6.1.xx-android14** GKI kernel
  (Test at your own risk!)
